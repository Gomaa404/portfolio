# Abdallah Mohamed Gomaa - Frontend Developer Portfolio

Welcome to the repository for my personal portfolio! This project showcases my skills, experience, and the projects I've worked on as a Frontend Developer. It's built with modern web technologies, emphasizing clean design, smooth animations, and high performance.

## 🚀 Live Demo
You can view the live portfolio here: https://one-three-nu.vercel.app/

## 🛠️ Technologies Used
- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons, Lucide React
- **Deployment:** Vercel

## ✨ Features
- **Responsive Design:** Fully optimized for both desktop and mobile devices.
- **Smooth Animations:** Integrated with Framer Motion for premium user interactions.
- **Theme Support:** Polished Dark and Light mode themes.
- **Dynamic Content:** Showcases real projects, skills, and experience dynamically.
- **Contact Form:** Easy way for recruiters or clients to reach out.

## 🏃‍♂️ Running Locally

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```
2. **Navigate into the project directory**
   ```bash
   cd one
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```
4. **Start the development server**
   ```bash
   npm run dev
   ```
